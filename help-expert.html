Cette interface en mode expert permet :


- d'explorer les données Naija via les opérations textométriques "classiques" : concordancier, spécificités, cooccurrents, etc.

- d'explorer les données Naija au niveau sociolinguistique.- d'explorer les relations de dépendance dans le corpus Naija

Une fois activé le mode expert, vous des boutons de partition qui permettent de mettre au jour un tableau donnant à voir pour tous les mots avec leur fréquence et leur indice de spécificité sur toutes les parties induites par la partition sélectionnée.



Cliquez sur l'icône ci-dessous pour afficher le mot en contexte dans un concordancier. Vous verrez ensuite une concordance du couple (lemme gouv, dep pos) visé (le gouverneur est le pôle de la concordance et son dépendant est surligné en jaune)



Dependency

Entrez une forme pôle puis cliquez sur le bouton Cooccurrent pour afficher un graphe de cooccurrents sur le pôle donné (avec retour en contexte).
