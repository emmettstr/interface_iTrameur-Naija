Cette interface permet :

- d'explorer les données Naija via les opérations textométriques "classiques" : concordancier, spécificités, cooccurrents, etc.

- d'explorer les données Naija au niveau sociolinguistique.

Requête

Une fois que la base est chargée, vous trouverez à gauche une série de boutons pour délimiter vos choix.

Cliquez sur des boutons qui vous intéressent et choisissez un critère par facette. i.e. SEX : F, AGE : < 15 si vous voulez observer l'indice de spécificité des mots chez les filles de moins de 15 ans. Puis cliquez sur le bouton Spécificités partie sélectionnée en bas pour générer le résultat induit par vos choix.

Lecture de résultat

Un nuage de mots disposé. Plus la taille de la police est grande, plus l'indice de spécificité du mot (sa sur-représentation dans les facettes choisies) est élevée.



Cliquez sur un mot pour afficher le mot en contexte dans un concordancier.

Son

Vous pouvez cliquer sur un mot pour écouter la prononciation de ce terme ou éventuellement l'échantillon complet.
